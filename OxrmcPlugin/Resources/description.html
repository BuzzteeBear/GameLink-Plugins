Instructions

  1. Set your desired motion limits (in °) as multiplier value for each axis under Motion & Vibration
  2. Launch COR Estimator App
  3. Start the plugin
  4. Launch your game (and optionally switch into VR)
  5. Switch over to COR Estimator in Desktop view and:
    1. Check and meet prerequisites
    2. Set options
    3. Start estimation process (optionally set a delay to get into position)
  6. During estimation keep your hmd (or motion controller) aligned with the moving rig
  7. Wait for estimation process to finish
  8. Optionally: Lock COR position within VR playspace (ctrl+home by default)
